# less 명령어로 파일 정보 보기

텍스트 파일 내용을 확인



```
less filename

less /etc/passwd
```



less : 텍스트 파일 내용 조회

| 항목                      | 의미                                        |
| ------------------------- | ------------------------------------------- |
| PAGE   UP 또는 b          | 한페이지   위로                             |
| PAGE   DOWN 또는 SPACEBAR | 한페이지   아래로                           |
| 위쪽   방향키             | 한줄   위로                                 |
| 아래쪽   방향키           | 한줄   아래로                               |
| G                         | 텍스트   파일 맨 처음으로 이동              |
| 1G   또는 g               | 텍스트   파일 맨 처음으로 이동              |
| /문자열                   | 아래   방향으로 진행하며 입력된 문자열 찾기 |
| n                         | 이전   검색어의 다음 찾기                   |
| h                         | 도움말   보기                               |
| q                         | 프로그램   종료                             |

   

리눅스 시스템 디렉토리 #1



| 항목        | 의미                                                         |
| ----------- | ------------------------------------------------------------ |
| /           | 루트디렉토리                                                 |
| /bin        | 시스템   부팅과 실행에 필요한 바이너리(프로그램)을 포함      |
| /boot       | 리눅스   커널 시작 RAM디스크 이미지 부트로더 포함   (/boot/vmliunuz:   리눅스 커널) |
| /dev        | 디바이스   노드 “모두 다 파일이다.” 디바이스도 파일로 관리   |
| /etc        | 시스템   전반의 환경 설정 파일                               |
| /home       | 각   사용자의 /home 디렉토리.                                |
| /lib        | 주요   시스템 프로그램에서 사용하는 공유 라이브러리 파일 저장 |
| /lost+found | 포맷된   각 파티션이나 ext3처럼 리눅스 파일 시스템에서 사용하는 디바이스라면 존재 |
| /media      | 최신   리눅스 시스템 CD-ROM, USB 드라이버 등이 여기에 마운트 된다. |
| /mnt        | 구형   리눅스 시스템에서 /mnt 마운트 포인트를 저장           |





| 항목           | 의미                                                         |
| -------------- | ------------------------------------------------------------ |
| /opt           | 추가적인   소프트웨어를 설치 할 때 사용                      |
| /proc          | 커널이   관리 하는 가상 파일 시스템                          |
| /root          | 루트   계정의 홈 디렉토리                                    |
| /sbin          | 시스템   바이너리 파일                                       |
| /tmp           | 임시   저장용 디렉토리                                       |
| /usr           | 일반   사용자가 사용하는 모든 프로그램과 지원 파일들이 들어있다. |
| /usr/bin       | 리눅스   배포판이 설치한 실행 프로그램들이있다.              |
| /usr/lib       | /usr/bin   디렉토리에 있는 프로그램을 위한 공유 라이브러리가 있다. |
| /usr/local     | 일반적으로   사용하는 배포판 프로그램들 대신 시스템 전반에 사용되는 프로그램들의 저장공간 |
| /usr/local/bin | 소스코드로   컴파일된 프로그램이 설치된다.                   |







| 항목           | 의미                                                         |
| -------------- | ------------------------------------------------------------ |
| /usr/sbin      | 시스템   관리 프로그램들                                     |
| /usr/share     | /usr/bin   디렉토리의 프로그램이 사용하는 공유데이터 저장    |
| /usr/share/doc | 시스템에   설치된 대부분의 패키지의 문서                     |
| /var           | 상대적으로   변하기 쉬운 데이터를 모아놓은 공간.   다양한   데이터베이스, 스풀 파일, 사용자 메일 등과 같은 데이터가 저장된다. |
| /var/log       | 시스템   활동을 기록하는 로그 파일이 들어있다.               |
