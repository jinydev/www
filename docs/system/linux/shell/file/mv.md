# 파일이동
---


## mv - 파일 이동과 이름 변경

mv item1 item2

mv item... directory



| 옵션                | 설명                                                         |
| ------------------- | ------------------------------------------------------------ |
| -i,   --interactive | 기존   파일을 덮어쓰기 전에 확인 메시지를 보여주는 옵션   (mv   명령어를 사용하면 확인 과정 없이 그대로 파일을 덮어쓰게 된다.) |
| -u,   --update      | 파일을   다른 디렉토리로 이동할때 최신 버젼의 파일만 이동하기 위해 사용하는 옵션 |
| -v,   --verbose     | 이동이   완료되었을때 메시지를 보여준느 옵셥                 |



| 사용   예제         | 결과                                                         |
| ------------------- | ------------------------------------------------------------ |
| mv file1 file2      | file1   파일을 file2 파일로 이동 (rename)   (file2   파일이 있다면 file1 내용을 그대로 덮어쓴다. file2 가 없다면 새로 생성) |
| mv -i file1 file2   | file2   파일이 있다면 사용자에게 덮어쓰기 여부를 확인한다.   |
| mv file1 file2 dir1 | file1   과 file2 를 dir1 이라고 하는 디렉토리로 이동         |
| mv dir1 dir2        | dir1과   그 내용까지 모두 dir2 로 이동 (move)                |
