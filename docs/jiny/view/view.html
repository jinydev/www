뷰(view)

지니PHP 의 MVC패턴의 view를 처리하는 패키지 입니다. 뷰는 컨트롤러, 라우트에 의해서 호출되어 지게 됩니다.

http response는 반환된 뷰의 화면을 브라우저로 전송하고, 브라우저는 전송받은 데이터를 기준으로 화면을 렌더링 하게 됩니다.

뷰 설치

view는 별도의 페키지로 분리되어 있습니다. 또한 뷰 페키지는 지니PHP에 기본적으로 설치되는 페키지 입니다. 설치가 안되어 있다면 다음과 같이 컴포저를 통하여 재설치 할 수 있습니다.

composer require jiny/view

지니PHP의 뷰를 다른곳에서도 응용해서 설치를 할 수 있습니다. 또는 인터페이를 맞추어 다른 프레임워크의 뷰로 교체를 할 수도 있습니다.
