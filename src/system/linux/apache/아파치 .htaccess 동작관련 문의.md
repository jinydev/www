아파치 .htaccess 동작관련 문의

아파치 웹서버의 rewriteRule 을 활성하 해서 사용하고자 합니다.
/home/apache/conf/extra/httpd-vhost.conf 에서

<VirtualHost *:80>
    DocumentRoot /home/wwwhtml/erp1.sales
    ServerName erp1.saleshosting.co.kr
    RewriteEngine On
    RewriteRule ^/([A-Za-z0-9_]+)/?$ /index.php?route=$1&v1=$2
    RewriteRule ^/([A-Za-z0-9_]+)/([A-Za-z0-9_]+)/?$ /index.php?route=$1&v1=$2
</VirtualHost>

처럼 직접 작성을 하여 서버를 시작하면 
http://erp1.saleshosting.co.kr/goodlist/1 접속이 잘 되어 집니다.

==> 
아파치 설정을 아래와 같이 수정하고
<VirtualHost *:80>
    DocumentRoot /home/wwwhtml/erp1.sales
    ServerName erp1.saleshosting.co.kr
</VirtualHost>

해당 Rewrite부분을 .htaccess 파일에 
/home/wwwhtml/erp1.sales/ .htaccess 에 파일에 

RewriteEngine On
RewriteRule ^/([A-Za-z0-9_]+)/?$ /index.php?route=$1&v1=$2
RewriteRule ^/([A-Za-z0-9_]+)/([A-Za-z0-9_]+)/?$ /index.php?route=$1&v1=$2

로 작성하여 동일하게 
http://erp1.saleshosting.co.kr/goodlist/1 접속시 해당 rewrireRule이 처리가 되지 않습니다.
어떻게 설정을 수정해야 되지 몰라서 이렇게 환경설정 방법에 대해서 문의드립니다.
